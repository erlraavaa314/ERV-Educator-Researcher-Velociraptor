Courses

I'm usually involved with the following courses at HVL:

  • DAT102: Algorithms and Data Structures
    Your basic algorithms and data structures course. You will learn how to think properly about the code you are writing, how to tell good code from bad and it really helps you pass coding interviews when you will apply for jobs.
  • DAT108: Programming and Web Applications
    An introductory course on web development. This is a super useful course where you will learn about both frontend and backend. We will also touch on central concepts like functional programming and paralell programming.
  • DAT158: Machine Learning and Algorithms
    A hands-on introduction to the machine learning (aka. AI on twitter) wave that's all around us. In addition to diving into algorithms, you will learn about generative AI and how to train your own classification models.
  • DAT191: Bachelor Thesis
    This is probably going to be the best and most demanding course of your bachelor degree. Here you get the opportunity to explore any topic of your choosing and/or to make something new completely from scratch! If you're interested in having me as your advisor, see my mentoring page or just contact me directly.
